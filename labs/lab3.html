To-Do List App Development

This lab involved the development of a fully functional To-Do List application using Android Studio. The app features task creation, editing, marking as completed, and deletion, along with seamless navigation between multiple screens.

Key Topics Covered:

  • Creating dynamic UI elements using XML layout files.
  • Implementing multi-screen navigation with explicit intents.
  • Integrating CRUD operations for task management.

Key Features of the App:

  • Add new tasks with custom configurations.
  • Mark tasks as completed or delete them.
  • Navigate seamlessly between screens for task details and configuration.

Challenges Faced:

  • Initial attempt to create a math quiz app faced numerous crashes and debugging challenges, leading to a redesign.
  • Resolved issues related to activity transitions and maintaining data consistency across screens.
  • Debugged complex XML layout errors to achieve desired UI responsiveness.

Interesting Insights:

  • Learned to use Android Studio's emulator and external emulators like BlueStacks for comprehensive testing.
  • Enhanced understanding of Android's lifecycle methods to optimize app performance and prevent crashes.

Reflection:

Transitioning from the initial math quiz app to the To-Do List app was a significant learning experience. While challenging, the lab helped improve my debugging skills and deepened my understanding of Android Studio's tools and capabilities. Overall, this experience strengthened my confidence in mobile app development.
