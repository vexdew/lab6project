App Development Basics

This classwork involved developing a personalized Android app that displayed essential details such as the developer's name, Banner ID, favorite quote, and class details. The app utilized Android Studio and focused on layout design and text customization.

Key Topics Covered:

  • Creating and customizing layouts using Android Studio's ConstraintLayout.
  • Personalizing UI elements with fonts, colors, and text alignment.
  • Testing app functionality using Android Studio Emulator and external emulators like BlueStacks.

App Features:

  • Displays developer details, including:
    • Name: Andrei Koshelev
    • Banner ID: 100894383
    • Class: INFT 3101-02
    • Favorite Quote: "I'd rather be hated for who I am, than loved for who I am not." ― Kurt Cobain
  • Utilizes a custom background image for an enhanced aesthetic appeal.
  • Ensures text and UI responsiveness across different screen sizes.

Challenges Faced:

  • Configuring the background image in the Android Studio environment, which required troubleshooting layout settings.
  • Long loading times for the Android Studio Emulator, resulting in higher system resource usage.
  • Manually saving and inserting emulator screenshots into the report due to formatting issues.

Reflection:

This classwork was an excellent introduction to Android app development. Despite minor challenges, the process was enjoyable, and the user-friendly interface of Android Studio made coding and designing straightforward. I look forward to creating more advanced apps and exploring additional functionalities in future assignments.
