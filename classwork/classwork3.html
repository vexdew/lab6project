Flutter App Development

This classwork involved building a personalized Flutter app to showcase basic personal information, including name, Banner ID, class details, and a favorite quote. The app utilized the Flutter SDK, Material Design components, and external emulators for testing.

Key Topics Covered:

  • Setting up the Flutter SDK and configuring it with Visual Studio Code and Android Studio.
  • Building a personalized app using Flutter's widget-based architecture.
  • Testing the app on both Android Studio Emulator and BlueStacks.

App Features:

  • Displays key personal information, including:
    • Name: Andrei Koshelev
    • Banner ID: 100894383
    • Class: INFT 3101-02
    • Favorite Quote: "I would rather be hated for who I am, than loved for who I am not." - Kurt Cobain
  • Customizable UI with a background image and styled text elements.
  • Responsive design that adapts to various screen sizes.

Challenges Faced:

  • Initial configuration of the Flutter SDK on the system and setting up emulators.
  • Resolving dependency conflicts and debugging errors during setup.
  • Long boot times for the Android Studio Emulator, requiring optimization for testing efficiency.

Reflection:

This project offered valuable experience with Flutter's framework, highlighting its flexibility and ease of use for building cross-platform applications. While challenging, the troubleshooting process was rewarding, and comparing the workflows of Flutter and Android Studio provided deeper insights into mobile app development environments. Overall, I enjoyed learning Flutter, and this classwork solidified my foundational knowledge for more complex projects in the future.
